### Installation
 - `git clone https://github.com/kevinbowen777/django_api-todo.git`
 - `cd django_api-todo`
 - `docker-compose up --build`
 - `docker-compose exec backend python backend/manage.py migrate`
 - URLs:
     - http://127.0.0.1:8000/api
     - http://127.0.0.1:3000/
